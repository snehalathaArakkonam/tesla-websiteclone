## ✨ Project Overview

This project is a **highly accurate clone of the Tesla India homepage**, designed to replicate the premium UI/UX experience of Tesla’s official website.

🔹 Built using **pure frontend technologies**  
🔹 Focused on **performance, responsiveness, and clean design**  
🔹 Structured as a **single-file production-ready application**

---

## 🚀 Features

- 🟢 Pixel-perfect Tesla UI replication  
- 🟢 100% responsive (Mobile / Tablet / Desktop)  
- 🟢 Sticky transparent navbar with scroll effects  
- 🟢 Smooth scrolling & transitions  
- 🟢 Real Tesla images (no placeholders)  
- 🟢 Mobile-friendly hamburger menu  
- 🟢 Clean modern UI with micro-interactions  
- 🟢 Fast loading & optimized structure
